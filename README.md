# 思维模型编辑器

一个用于管理和编辑思维模型的可视化编辑器，支持508个思维模型的浏览、编辑和保存。

## 功能特性

- **可视化编辑**：直观的界面，支持实时预览
- **508个思维模型**：涵盖商业、心理学、科学等多个领域的思维模型
- **拟态设计**：采用Neumorphism设计风格，美观现代
- **主题切换**：支持日间/夜间模式切换
- **搜索功能**：快速查找特定思维模型
- **实时预览**：编辑内容时实时显示预览效果

## 使用方法

1. 双击 `启动思维模型编辑器.bat` 文件一键启动（推荐）
2. 或者双击 `optimizedModelEditor.html` 文件在浏览器中打开
3. 通过顶部搜索框搜索特定模型
4. 在模型列表中点击选择要编辑的思维模型
5. 在右侧编辑区域修改模型内容
6. 点击右上角"保存更改"按钮保存修改

## 界面布局

- **顶部**：搜索框和主题切换按钮
- **中部**：模型选择区域，显示所有思维模型按钮（带序号）
- **底部**：编辑区域，分为左侧编辑区和右侧预览区

## 技术特点

- 纯HTML/CSS/JavaScript实现
- 无需后端服务，可离线使用
- 响应式设计，适配不同屏幕尺寸
- 拟态设计风格，提供现代化视觉体验

## 文件结构

- `optimizedModelEditor.html` - 主编辑器文件
- `thinkingModels.json` - 思维模型数据文件
- `启动思维模型编辑器.bat` - 一键启动脚本

## 数据结构

每个思维模型包含以下字段：

- `id` - 模型唯一标识
- `name` - 模型名称
- `label` - 显示标签
- `icon` - 图标名称
- `description` - 模型简介
- `deepAnalysis` - 深度分析
- `principle` - 核心原则
- `scope` - 适用范围
- `tips` - 使用技巧
- `practice` - 实践建议
- `scenario` - 应用场景
- `visualDesign` - 可视化设计
